# assemble-index-json

> assemble index generator


## Install
```
$ npm i --save-dev assemble-index-json
```

## Register
```
assemble: {
  options: {
    plugins: ['assemble-index-json']
  }
}
```

## options

### jsonPath

Path to a json is generated from this plugin.

### availables

Data list that you want to retrieve from page data.
